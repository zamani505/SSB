سهند سامانه برتر SSB

این َAPI ارسال پیامک به روش های مختلف و دریافت پیامک های ارسال شده و همچنین شارژ کاربر فراهم می کند

API: http://SSB/v1/api/SendSMS/{سرویس ارسال پیامک}
API: http://SSB/v1/api/Recieve/{سرویس دریافت پیامک}
API: http://SSB/v1/api/Credit/{سرویس دریافت شارز}
نسخه: 1.0.0

مستندات کامل API

خلاصه

API شرکت سهند سامانه برتر دارای انواع متد های ارسال و همچنین دریافت پیامک های ارسال شده که هربار امکان دریافت 100 پیامک می باشد و همچنین شما می توانید قبل ارسال پیامک میزارن شارژ مورد استفاده برای ارسال مدنظر و شارژ کاربری خود را دریافت نمایید

در تمامی سرویس در صورت بروز خطا کد مربوط به خطا در فیلد Code , شرح مختصر خطا در Message قرار میگیرد.API با بروز Exception به هیچ عنوان متن خطا را به کاربر نمایش نمیدهد و فقط در خروجی Code خطا را قرار میدهد

شروع سریع — مثال curl

curl --location 'http://SSB/api/SendSMS/SendPostUrl' \
--header 'SSBToken: YOURTOKEN' \
--header 'Content-Type: application/json'
data '{
    
        "message":"d",
        "fromNumber":"3000",
        "toNumber":"9127688"
    
}'

نمونه پاسخ (JSON)

{
"result":[],
"message":"",
"code":"0"
}

در صورت انتشار نسخه جدید فقط کافی است برای درسافت سرویس جدید ورژن Api که در Url قرار دارد را تغییر دهیدhttp://SSB/v1/api/Credit/ مانند v2/api/.
